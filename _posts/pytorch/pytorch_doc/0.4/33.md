

# torchvision

`torchvision`包朗阔了目前流行的数据集，模型结构和常用的图片转换工具。

如下代码用于获取加载图像的包的名称。

```py
torchvision.get_image_backend()
```

指定用于加载图像的包。

```py
torchvision.set_image_backend(backend)
```

参数：
backend(string）-backend代表图片名称。属于{'pil ','accimage'}之一。accimage包使用Intel IPP库。速度比 PIL快，但不支持多操作。
